Búsqueda Tabú

La Búsqueda Tabú es un algoritmo metaheurístico que utiliza memoria para evitar ciclos y explorar mejor el espacio de búsqueda. Mantiene una lista de movimientos recientes (lista tabú) que están prohibidos temporalmente, forzando al algoritmo a explorar nuevas regiones. Esto permite escapar de óptimos locales sin depender de aleatoriedad como en simulated annealing.

Vecindario: El algoritmo utiliza conectividad-8, considerando los 8 vecinos adyacentes. En cada iteración, selecciona el mejor vecino no tabú, o aplica el criterio de aspiración si un movimiento tabú mejora la mejor solución encontrada.

Colores: ● Azul = Posición actual | ● Verde = Visitado | ● Naranja = Vecino considerado | ● Rojo = Movimiento tabú | ● Morado = Mejor solución | ● Amarillo = Criterio de aspiración
