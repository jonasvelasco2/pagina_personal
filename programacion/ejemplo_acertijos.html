📚 Paso 1: El Acertijo de la Entrada

Para entrar a la biblioteca, Robi debe resolver un acertijo que combina patrones y lógica. El Sabio muestra una secuencia de símbolos y Robi debe completar el patrón.

🧩 Acertijo: "Si el patrón es ⭐🔺⭐🔺, ¿qué sigue?"

💡 Solución: Robi identifica el patrón (estrella-triángulo) y usa un BUCLE para completar la secuencia.

Biblioteca Mágica 4x4

🧠 Algoritmo Complejo:

1. PLANIFICAR ruta a cada estante
2. PARA cada estante:
• SI estante tiene libro correcto
→ ENTONCES tomar libro
• REPETIR hasta encontrar todos
3. ENTREGAR libros en orden específico

📖 Paso 2: La Búsqueda del Tesoro de Conocimiento

El Sabio pide a Robi que encuentre 5 libros específicos escondidos en diferentes estantes. Cada libro tiene pistas sobre dónde está el siguiente, y algunos estantes están protegidos por acertijos.

🧠 Desafío: Combinar planificación (ruta óptima), condicionales (verificar libros) y bucles (buscar sistemáticamente).

🌟 Paso 3: El Acertijo Final del Sabio

El último desafío combina TODOS los conceptos en un meta-acertijo: Robi debe programar a otros robots virtuales para que resuelvan laberintos, recojan cristales, eviten obstáculos y construyan puentes... ¡todo al mismo tiempo!

📝 Secuencia de instrucciones para cada robot
🔁 Bucles para tareas repetitivas
🤔 Condicionales para decisiones inteligentes
🗺️ Planificación para coordinar todos los robots
