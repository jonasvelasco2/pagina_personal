¿Cómo se construye la curva ROC?

La curva ROC muestra la relación entre la tasa de verdaderos positivos (TPR) y la tasa de falsos positivos (FPR) para diferentes umbrales de clasificación. Cada punto de la curva corresponde a un umbral diferente. El área bajo la curva (AUC) resume el rendimiento del clasificador: AUC = 0.5 es aleatorio, AUC = 1.0 es perfecto.
