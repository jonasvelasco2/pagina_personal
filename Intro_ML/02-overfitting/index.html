¿Qué estamos observando?

Dibuja puntos de dos clases diferentes y entrena un modelo para ver cómo separa las clases. Con complejidad baja, el modelo será muy simple (subajuste). Con complejidad alta, el modelo intentará memorizar cada punto (sobreajuste).
