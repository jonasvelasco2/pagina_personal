¿Por qué usar validación cruzada?

La validación cruzada permite utilizar todos los datos tanto para entrenar como para evaluar, obteniendo una estimación más robusta del error de generalización. Con K-fold, dividimos los datos en K partes iguales, y en cada iteración usamos K-1 partes para entrenar y 1 parte para validar.
