Función sigmoide y frontera de decisión

Explora cómo la función sigmoide transforma valores lineales en probabilidades y cómo los pesos definen la frontera de decisión en regresión logística.

Función sigmoide: σ(z) = 1 / (1 + e⁻ᶻ)

z = 0.00 → σ(z) = 0.500

Interpretación: La función sigmoide comprime cualquier valor real z ∈ (-∞, +∞) al intervalo (0, 1), permitiendo interpretar la salida como una probabilidad.

• Si z → -∞, entonces σ(z) → 0
• Si z = 0, entonces σ(z) = 0.5
• Si z → +∞, entonces σ(z) → 1

La frontera de decisión es donde z = 0, es decir: 1.00·x₁ + 1.00·x₂ + 0.00 = 0
